# TaBlazor
### It's Tabler... for Blazor!

[![License][badge-license]][badge-license-url]
![Supports .NET 6.0][badge-net-6]
![Supports .NET 7.0][badge-net-7]
![Open Issues][badge-issues]

TaBlazor is a Razor Class Library (RCL) containing Blazor Components 
based entirely on the [Tabler CSS Framework][tabler-homepage] for the UI.

TaBlazor is **heavily** inspired by some awesome projects.  Parts of TaBlazor are based
off of these projects.

[MudBlazor][mudblazor-homepage] • [Blazorize][blazorize-homepage]

My goal is to make it as easy as possible without sacrificing on customization
and control.
